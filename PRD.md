# 遊戲中心 PRD

## 專案概述
Game On: MiniVerse 是一個純前端網頁遊戲平台，提供三款不同類型的互動遊戲。專案採用現代網頁技術（HTML5、CSS3、JavaScript）實現，無需任何框架或建置工具。設計風格為深色科技感玻璃擬態，提供流暢的使用者體驗與響應式設計。

**核心目標**：建立一個教育性質的遊戲展示平台，展示現代網頁開發技術與遊戲設計能力。

## 基本資訊

- **網站名稱**：Game On: MiniVerse
- **設計風格**：深色科技感 (Dark Tech Aesthetic) 玻璃擬態設計
- **主色調**：深藍黑 (#05070a) + 霓虹藍 (#00f2ff) + 紫色 (#7000ff) + 粉紅 (#ff00c8)

## 遊戲列表

1. **極速賽車 (卡通賽道賽車)** - 體驗腎上腺素飆升的極速競速，在卡通風格賽道上駕駛跑車
   - 檔案：`game1.html`
   - 類型：競速、動作、單人
   - 技術：Canvas 2D 渲染、鍵盤控制 (方向鍵)、碰撞檢測、遊戲循環
   - 控制方式：方向鍵控制車輛移動，避開障礙物

2. **井字過三關** - 經典的井字遊戲，考驗你的策略思維與預判能力
   - 檔案：`game2.html`
   - 類型：策略、棋盤、雙人
   - 技術：DOM 操作、遊戲狀態管理、勝利條件算法、動畫效果
   - 遊戲模式：玩家對玩家 (PvP)，輪流放置 X 和 O

3. **天下太平 (飛機大砲遊戲)** - 傳統的猜拳策略遊戲，結合堡壘建設與攻擊機制
   - 檔案：`game3.html`
   - 語言：繁體中文 (zh-HK)
   - 類型：策略、猜拳、雙人
   - 技術：猜拳機制、回合制策略、堡壘管理、資源系統
   - 遊戲機制：通過猜拳決定行動權，建設堡壘或攻擊對手

## 設計規範

### 色彩系統 (CSS 變數定義)
- **背景色**：深藍黑 (#05070a) `--bg-dark`
- **主要強調色**：霓虹藍 (#00f2ff) `--accent-primary`
- **次要強調色**：紫色 (#7000ff) `--accent-secondary`
- **第三強調色**：粉紅 (#ff00c8) `--accent-tertiary`
- **主要文字**：淺藍灰 (#e2e8f0) `--text-main`
- **次要文字**：灰藍色 (#94a3b8) `--text-dim`
- **玻璃擬態背景**：半透明白色 (rgba(255, 255, 255, 0.03)) `--glass`
- **玻璃邊框**：半透明白色 (rgba(255, 255, 255, 0.1)) `--glass-border`
- **卡片陰影**：深色陰影 `--card-shadow`

### 字體系統
- **標題字體**：Orbitron (科技感無襯線字體)
- **正文字體**：Noto Sans TC (繁體中文支援)
- **字體大小**：
  - 桌面：標題 4.5rem，正文 1rem
  - 手機：標題 3rem，正文 0.9rem

### 動畫效果
1. **卡片懸浮效果**：hover 時上移 10px + 輕微放大
2. **光澤掃過效果**：卡片內光澤動畫
3. **圖標旋轉**：hover 時圖標輕微旋轉
4. **漸入動畫**：頁面載入時淡入效果

## 技術規格

### 前端技術棧
