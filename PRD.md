## 功能需求

### 核心功能
1. 遊戲導覽頁面（index.html）展示所有遊戲卡片
2. 每個遊戲卡片點擊後進入對應遊戲頁面
3. 響應式設計，支援手機、平板、桌面
4. 繁體中文介面

### 進階功能
1. 黑暗/明亮模式切換
   - 提供深色科技感與明亮可愛風兩種主題
   - 使用者可透過切換按鈕即時變更
   - 記住使用者的主題偏好（localStorage）
2. 遊戲進度保存
3. 音效開關控制
4. 遊戲分數排行榜

### 主題切換實現方案
1. 在 `common.css` 中定義兩組 CSS 變數：
   ```css
   :root {
     /* 深色科技感主題 */
     --bg-dark: #0a0a0f;
     --bg-card: #1a1a2e;
     --text-main: #ffffff;
     --accent-primary: #00f2ff;
     /* ... */
   }

   [data-theme="light"] {
     /* 明亮可愛風主題 */
     --bg-dark: #f8f9fa;
     --bg-card: #ffffff;
     --text-main: #333333;
     --accent-primary: #ff6b8b;
     /* ... */
   }
   ```
2. 透過 JavaScript 切換 `document.documentElement.dataset.theme`
3. 將主題偏好儲存在 `localStorage`

## 設計規範

### 色彩系統
- **深色科技感**：
  - 背景：深藍黑 (#0a0a0f)
  - 卡片：深紫藍 (#1a1a2e)
  - 強調色：霓虹藍 (#00f2ff)
  - 文字：白色 (#ffffff)

- **明亮可愛風**：
  - 背景：淺灰 (#f8f9fa)
  - 卡片：白色 (#ffffff)
  - 強調色：粉紅 (#ff6b8b)
  - 文字：深灰 (#333333)

### 排版
- 標題字體：Orbitron（科技感）
- 正文字體：Noto Sans TC（繁體中文支援）
- 字體大小：響應式設計，桌面 16px，手機 14px

### 動畫效果
- 卡片懸浮效果（translateY + scale）
- 按鈕點擊反饋
- 主題切換過渡動畫（0.3s ease）
